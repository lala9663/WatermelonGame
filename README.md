# WatermelonGame
최근 핫 한 수박게임 만들기

## 설치 방법

```
1. node 설치
2. git clone https://github.com/lala9663/WatermelonGame.git
3. cd WatermelonGame
4. npm install
```

## 실행 방법

```
npn run dev
```

## 게임 설명

해당 게임은 같은 과일 2개가 만나면 다음 좀 더 큰 과일로 변하며 그 과일 2개가 다시 만다면 조금 더 큰 과일로 만나는 게임입니다.  
최종적으로 수박 2개를 만드는 게임입니다. 

    
방향키 : W(<-), S(내리기), D(->)
